# Portfolio Setup

A personal portfolio built to showcase my work in business strategy, product analytics, and full-stack development. Designed to be fast, minimal, and decision-focused — just like the work it presents.

🌐 **Live:** [portfolio-roan-nine-78.vercel.app](https://portfolio-roan-nine-78.vercel.app)

---

## About

I started in computer science because I loved building systems. Over time, I realized the hardest part isn't code — it's deciding what to build, what to ignore, and how to make decisions under constraints.

This portfolio reflects that thinking: clean architecture, purposeful design, and content built around outcomes — not features.

---

## Stack

| Layer | Tech |
|---|---|
| Framework | React 19 + TypeScript |
| Build Tool | Vite |
| Styling | Tailwind CSS v4 |
| Icons | Lucide React |
| Hosting | Vercel |

---

## Features

- Scroll-based reveal animations
- Live ticker — ongoing learning & projects
- Certificate lightbox with blur backdrop
- Persistent view counter (localStorage)
- Google Form contact integration
- Fully responsive — mobile + desktop
- Resume download
